NVIDIA Blackwell B200 공급 확대 — TSMC CoWoS-L 패키징 월 5만 장 돌파
TSMC가 CoWoS-L 고대역폭 패키징 월 생산량을 5만 장까지 늘리며 NVIDIA Blackwell GPU 공급 병목이 완화되고 있다. 2분기 B200 출하량은 전 분기 대비 38~42% 증가할 것으로 업계는 추산한다. 다만 마이크로소프트·메타·구글의 선점 계약으로 신규 주문사 납기는 여전히 6~9개월 대기다.
2026-03-18 · Tom's Hardware / DigiTimes 주요매체
CoWoS-L—TSMC의 Chip-on-Wafer-on-Substrate 고급형 패키징 기술. HBM 메모리와 GPU 다이를 하나의 기판에 통합해 대역폭을 극대화. Blackwell GPU 핵심 제조 공정.
HBM (High Bandwidth Memory)—GPU에 탑재되는 고대역폭 메모리. DRAM을 수직으로 쌓아 초당 수 TB 데이터 처리. AI 학습·추론 속도를 좌우하는 핵심 부품으로 SK하이닉스·삼성이 주 공급사.
TCO (Total Cost of Ownership)—총 소유 비용. 구매가에 전력·냉각·유지보수 비용을 더한 실제 운용 비용. AI 칩 도입 결정에서 단순 가격보다 TCO를 비교하는 추세.
삼성전자, 온디바이스 AI 전용 NPU 'Mach-1' 아키텍처 상세 공개
삼성전자 DS부문이 갤럭시 S26에 탑재될 차세대 NPU 'Mach-1'의 아키텍처를 처음 공개했다. 70억~130억 파라미터 언어모델을 스마트폰에서 실시간 추론하는 데 특화됐으며, 동급 퀄컴 Hexagon 대비 TOPS/W(와트당 연산) 기준 28% 우위를 주장한다. 3nm 게이트-올-어라운드(GAA) 공정으로 생산되며 전력 소모는 2.1W로 설계됐다.
2026-03-18 · 전자신문 / The Elec 국내매체
NPU (Neural Processing Unit)—딥러닝 행렬 연산에 특화된 프로세서. CPU·GPU보다 AI 추론 효율이 높아 스마트폰·엣지 디바이스에 탑재. 삼성·퀄컴·애플(ANE) 등이 독자 개발 중.
GAA (Gate-All-Around)—3nm 이하 반도체 공정에서 사용하는 트랜지스터 구조. 게이트가 채널을 사방에서 둘러싸 전력 누설을 줄이고 성능을 높임. 삼성이 TSMC보다 먼저 양산 적용.
리벨리온·퓨리오사AI, KT·SKT 데이터센터 NPU 입찰 맞대결
국내 AI 반도체 스타트업 리벨리온과 퓨리오사AI가 KT·SKT 데이터센터 NPU 공급 입찰에 동시 참가했다. 두 회사 모두 NVIDIA H100 대비 총 소유 비용 35% 절감을 제안했으며, 연내 납품 가능한 국산 칩으로 공급망 리스크 해소를 내세웠다. 최종 결정은 4월 중으로 예상된다.
2026-03-17 · 전자신문 / 한국경제 국내매체
리벨리온 (Rebellions)—2020년 설립 국내 AI 반도체 스타트업. ATOM 칩으로 데이터센터용 추론 가속기 시장 공략. KT와 전략적 파트너십을 맺고 있으며 나스닥 상장 준비 중.
퓨리오사AI (FuriosaAI)—2017년 설립 국내 AI 반도체 스타트업. WARBOY·RNGD 칩으로 비전 AI·LLM 추론 최적화. 삼성벤처투자·SKT 등으로부터 투자 유치.
AMD MI350 출시 일정 3분기로 앞당겨 — NVIDIA 대항마 기대감
AMD가 MI350 AI 가속기 출시 시점을 당초 4분기에서 3분기로 앞당겼다고 밝혔다. CDNA 4 아키텍처 기반으로 MI300X 대비 추론 성능 2.4배, 메모리 대역폭 1.8배 향상이 예고됐다. Microsoft Azure가 미리보기 파트너로 참여하며 CUDA 대안 생태계인 ROCm 6.2를 함께 공개한다.
2026-03-18 · AnandTech / The Register 주요매체
ROCm—AMD의 오픈소스 GPU 컴퓨팅 플랫폼. NVIDIA의 CUDA에 대응하는 소프트웨어 스택으로, PyTorch·TensorFlow 등 주요 AI 프레임워크를 AMD GPU에서 실행 가능하게 함.
CDNA (Compute DNA)—AMD의 데이터센터 전용 GPU 아키텍처. 게이밍용 RDNA와 달리 행렬 연산·AI 학습에 최적화. MI 시리즈(MI300X, MI350)의 핵심 설계.
Groq, LPU 3세대 'GroqRack 3' 공개 — 와트당 추론 속도 업계 1위 주장
AI 추론 전문 칩 업체 Groq가 3세대 LPU(Language Processing Unit) 기반 GroqRack 3를 공개했다. Llama 3.1 70B 기준 초당 800토큰 생성 속도를 달성하며 GPU 기반 시스템 대비 와트당 35배 효율을 내세웠다. AWS 마켓플레이스를 통한 클라우드 서비스도 하반기 출시 예정이다.
2026-03-17 · SiliconANGLE / Groq 공식 주요매체
LPU (Language Processing Unit)—Groq가 개발한 LLM 추론 전용 칩. GPU와 달리 순차적 토큰 생성에 최적화된 데이터 흐름 아키텍처를 사용해 지연 시간이 극도로 낮음.
토큰 (Token)—LLM이 처리하는 텍스트의 최소 단위. 영어 단어 기준 약 0.75개 분량. 초당 토큰 수(tokens/sec)는 AI 추론 속도의 핵심 지표.

Anthropic, Claude Code 정식 출시 — SWE-bench 72.5% · 월 $100 플랜
Anthropic이 터미널에서 실행하는 AI 코딩 에이전트 'Claude Code'를 정식 출시했다. 실제 소프트웨어 엔지니어링 벤치마크인 SWE-bench Verified에서 72.5%를 기록, 경쟁 제품(GitHub Copilot 55%, Cursor 61%) 대비 확실한 우위를 보였다. 개인용 월 $100, 팀용 월 $500(5인) 플랜으로 구성됐으며, API 키 연결 시 종량제도 가능하다.
2026-03-18 · Anthropic 공식 / The Verge 주요매체
SWE-bench—AI 코딩 에이전트를 평가하는 표준 벤치마크. GitHub의 실제 이슈 2,294개를 얼마나 자동으로 해결하는지 측정. 숫자가 높을수록 실제 코드 작업 능력이 뛰어남.
AI 에이전트 (Agent)—단순 질문-답변을 넘어 파일 수정·명령 실행·웹 검색 등 연속 작업을 자율 수행하는 AI. Claude Code는 터미널에서 코드 작성→테스트→커밋까지 자동 처리.
종량제 (Pay-as-you-go)—사용한 만큼만 요금을 내는 방식. AI API에서는 입력·출력 토큰 수에 따라 과금. 사용량이 일정하지 않은 개발자·소규모 팀에 유리.
Meta, Llama 4 Scout 오픈 웨이트 공개 — 1090억 파라미터 MoE, Apache 2.0
Meta가 Llama 4 시리즈의 첫 번째 모델 'Scout'를 오픈 웨이트로 공개했다. 활성화 파라미터 170억 개의 MoE(전문가 혼합) 구조로 실제 연산량은 훨씬 적으면서도 1090억 파라미터급 성능을 낸다. Apache 2.0 라이선스로 상업 사용이 자유롭고, 공개 24시간 내 Hugging Face 다운로드 50만 회를 돌파했다.
2026-03-17 · Meta AI / Hugging Face 주요매체
MoE (Mixture of Experts)—모델 전체가 아닌 일부 '전문가' 레이어만 선택적으로 활성화하는 아키텍처. 전체 파라미터는 많지만 실제 연산은 적어 추론 속도와 효율이 높음. GPT-4, Gemini 1.5도 MoE 구조.
오픈 웨이트 (Open Weights)—모델의 가중치(파라미터 값)를 공개해 누구나 다운로드·배포·파인튜닝할 수 있는 방식. 소스코드 완전 공개인 '오픈소스'와 다르게 학습 데이터·코드는 비공개일 수 있음.
Google Gemini 2.5 Pro, 실시간 영상·3D 분석 기능 추가 — 제조·건설 현장 적용
Google DeepMind가 Gemini 2.5 Pro에 실시간 영상 스트림 분석과 3D 공간 이해 기능을 추가했다. 공장 컨베이어 벨트의 불량품을 실시간으로 감지하거나 건설 현장 도면과 현장 영상을 비교해 시공 오차를 자동 검출하는 데 활용된다. 국내 현대건설·삼성물산이 파일럿 프로젝트를 진행 중이다.
2026-03-18 · Google DeepMind / VentureBeat 주요매체
멀티모달 (Multimodal)—텍스트·이미지·오디오·영상 등 여러 형태의 데이터를 동시에 이해하고 생성하는 AI 능력. Gemini 2.5 Pro는 여기에 실시간 영상과 3D 공간 이해를 추가.
컨텍스트 윈도우—AI 모델이 한 번에 처리할 수 있는 최대 정보량. Gemini 2.5 Pro는 200만 토큰(책 약 1,500권 분량)으로 업계 최대 수준이며 긴 영상 분석에 유리.
오픈소스 추론 모델 전쟁 — DeepSeek R2·Qwen 3·Mistral Large 3 동시 공개
DeepSeek R2, 알리바바 Qwen 3-235B, Mistral Large 3가 동시에 공개되며 GPT-4o 수준의 추론 능력을 무료로 제공하는 오픈소스 경쟁이 격화됐다. DeepSeek R2는 수학·코딩 벤치마크에서 OpenAI o3를 근소하게 앞서며 충격을 줬고, Qwen 3-235B는 한국어 포함 29개 언어에서 강점을 보였다.
2026-03-16 · ArXiv / Hugging Face / GitHub 분석매체
추론 모델 (Reasoning Model)—답변 전 단계적 사고 과정(Chain-of-Thought)을 거치는 AI 모델. 수학·코딩·논리 문제에서 일반 모델보다 정확도가 높음. OpenAI o3, DeepSeek R1이 대표적.
벤치마크 (Benchmark)—AI 모델 성능을 표준화된 문제로 비교하는 테스트. MMLU(지식), HumanEval(코딩), MATH(수학) 등이 대표적. 단, 벤치마크에만 과적합하는 '굿하트의 법칙' 문제가 있음.
OpenAI Operator, 한국 포함 15개국 추가 출시 — 웹 자동화 에이전트 확장
OpenAI의 웹 자동화 에이전트 Operator가 한국을 포함한 15개국에 추가 출시됐다. 여행 예약·쇼핑몰 주문·정부 민원 신청 등 반복 웹 작업을 자율 수행한다. 국내 네이버·카카오 등 한국어 인터페이스 지원도 포함됐으나, 보안 우려로 금융 사이트 자동화는 제한됐다.
2026-03-18 · OpenAI 공식 / The Verge 주요매체
컴퓨터 사용 (Computer Use)—AI가 직접 마우스·키보드를 조작하듯 웹브라우저·애플리케이션을 제어하는 기술. Anthropic이 2024년 처음 공개하고 OpenAI Operator·Google Project Mariner 등이 경쟁 중.
웹 에이전트—사람 대신 웹사이트를 탐색·클릭·입력하는 AI. 단순 RPA(로봇 프로세스 자동화)와 달리 정해진 스크립트 없이 상황에 따라 판단하며 작업을 완수.
